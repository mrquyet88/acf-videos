== Description ==

Add file type Videos for ACF, You can choose video from Media.

= Compatibility =

This ACF field type is compatible with:
* ACF 5

== Installation ==

1. Copy the `acf-videos` folder into your `wp-content/plugins` folder
2. Activate the "ACF Videos" plugin via the plugins admin page
3. Create a new field via ACF and select the "Videos" type
4. Read the description above for usage instructions
